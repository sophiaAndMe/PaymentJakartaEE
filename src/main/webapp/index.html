BIENVENIDO A TU TUTI!!

insertar un cliente --> SuperMaxi/manager/create/name&address&payment&product&price&quantity

obtener todos los clientes --> SuperMaxi/manager/allCustomer

Borrar un cliente por id --> SuperMaxi/manager/delete/id

Actualizar el nombre y dirrecion --> SuperMaxi/manager/updateCustomer/id&name&address

Lista de cliente de un nombre --> SuperMaxi/manager/listName/name

Saber toda la informacion por nombre --> SuperMaxi/manager/allInformation/name
